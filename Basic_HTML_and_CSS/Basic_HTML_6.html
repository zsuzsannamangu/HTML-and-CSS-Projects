Learn about the HTML span tag

 The <span> tag is an inline container used to mark up only a part of a text, or a part of a document.
 The <span> tag is easily styled by CSS or manipulated with JavaScript using the class or id attribute.
 The <span> tag is much like the <div> element, but <div> is a block-level element and <span> is an inline element.

The <details> element in HTML

The <details> tag specifies additional details that the user can open and close on demand. The <details> tag is often used to create an interactive widget that the user can open and close. By default, the widget is closed. When open, it expands, and displays the content within.

The <time> element in HTML

The <time> tag defines a specific time (or datetime). The datetime attribute of this element is used translate the time into a machine-readable format so that browsers can offer to add date reminders through the user's calendar, and search engines can produce smarter search results.
