Learn about HTML entities

 Reserved characters in HTML must be replaced with character entities.
 Some characters are reserved in HTML.
 If you use the copyright (©) symbol in your text, the browser might not diplay it correctly.
 Character entities are used to display reserved characters in HTML.

Learn about the HTML span tag

 The <span> tag is an inline container used to mark up only a part of a text, or a part of a document.
 The <span> tag is easily styled by CSS or manipulated with JavaScript using the class or id attribute.
 The <span> tag is much like the <div> element, but <div> is a block-level element and <span> is an inline element.
